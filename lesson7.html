Lesson 7: Practice with Small Projects

Why Small Projects?

Working on small projects helps you solidify your Python knowledge by applying the concepts you've learned. These projects will build confidence and give you practical coding experience.

Project 1: Number Guessing Game

Create a simple game where the computer generates a random number, and the player tries to guess it. Here's how to approach it:

  • Use the random library to generate a random number.
  • Prompt the player to input their guess.
  • Give feedback if the guess is too high, too low, or correct.
  • Loop until the player guesses correctly or chooses to quit.
import random
number = random.randint(1, 100)
guess = None
while guess != number:
    guess = int(input("Guess a number between 1 and 100: "))
    if guess < number:
        print("Too low!")
    elif guess > number:
        print("Too high!")
    else:
        print("You guessed it!")

Project 2: Simple Calculator

Build a basic calculator that can perform addition, subtraction, multiplication, and division:

  • Ask the user for two numbers and an operator.
  • Perform the operation and display the result.
  • Use functions for each operation for modularity.
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    if y != 0:
        return x / y
    else:
        return "Cannot divide by zero!"

Project 3: Word Frequency Counter

Write a program that reads a text file, counts the frequency of each word, and displays the results:

  • Use with open() to read the file.
  • Use a dictionary to store word counts.
  • Display the most frequent words in the text.
word_count = {}
with open("sample.txt", "r") as file:
    for line in file:
        words = line.split()
        for word in words:
            word = word.lower().strip(",.!?")
            if word in word_count:
                word_count[word] += 1
            else:
                word_count[word] = 1
print(word_count)
