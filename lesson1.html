Lesson 1: Introduction to Python

What is Python?

Python is a high-level, interpreted programming language known for its easy-to-read syntax and powerful libraries.

Setting Up Python

To get started with Python, you first need to install it. You can download Python from the official website:

Download Python

Option 2: Using Google Colab

If you prefer not to install Python on your computer, you can use **Google Colab** to write and run Python code directly in your browser. It's a cloud-based Python environment that comes with many useful libraries pre-installed!

Start Using Google Colab

Your First Python Program

Once Python is installed (or if you're using Google Colab), you can write your first Python program:

print("Hello, Python!")

This program prints the text "Hello, Python!" to the screen. Try it out in your Python environment or Google Colab!

Basic Syntax

  • Python is case-sensitive, so print is different from Print.
  • Every Python program starts from the first line of code and executes line-by-line.
  • Comments can be added using the # symbol for single-line comments.

Exercise

Try writing your own Python program. Print your name using the print() function.

Good luck, and happy coding!
