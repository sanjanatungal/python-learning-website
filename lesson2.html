Lesson 2: Variables and Data Types

What is a Variable?

A variable is a container for storing data values. In Python, you do not need to declare a variable type. Python automatically assigns the type based on the value assigned to the variable.

Declaring Variables

You can assign a value to a variable by using the equals sign (`=`).

age = 25  # This is an integer variable
name = "John"  # This is a string variable

Common Data Types in Python

Python has various built-in data types, which include:

  • Integers (int): Whole numbers.
  • Floats (float): Decimal numbers.
  • Strings (str): Text data, enclosed in quotes.
  • Booleans (bool): Represents True or False values.

Examples

Here are examples of each data type in Python:

Integers

age = 25  # This is an integer
print(age)  # Output: 25

Floats

height = 5.9  # This is a float
print(height)  # Output: 5.9

Strings

name = "John"
print(name)  # Output: John

Booleans

is_student = True
print(is_student)  # Output: True
