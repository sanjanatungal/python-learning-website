Lesson 6: Get Hands-On with File I/O

What is File I/O?

File I/O (Input/Output) in Python refers to working with files for reading data from them or writing data to them. Python provides built-in functions to handle files easily.

Opening a File

To open a file in Python, you use the open() function. You can specify the mode you want to open the file in, such as "r" for reading, "w" for writing, or "a" for appending.

file = open("example.txt", "r")  # Opens a file for reading

Reading from a File

Once a file is opened in read mode, you can use the read(), readline(), or readlines() methods to read its contents.

content = file.read()  # Reads the entire file
print(content)

Writing to a File

To write to a file, you open it in "w" mode, which will overwrite the file if it already exists. You can also use "a" mode to append to the file without overwriting.

file = open("example.txt", "w")  # Opens a file for writing
file.write("Hello, world!")  # Writes to the file
file.close()  # Closes the file

Closing a File

It is essential to close a file after you are done with it to free up system resources. You can close a file using the close() method:

file.close()

Using with for File Handling

The with statement is a better way to work with files, as it automatically handles closing the file:

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

Exercise: Practice with File I/O

Try the following exercises to practice file handling:

  • Create a file called my_file.txt and write some text to it.
  • Open my_file.txt and read its contents, then print them to the console.
  • Append a new line to my_file.txt and verify it was added by reading the file again.
