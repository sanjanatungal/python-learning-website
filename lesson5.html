Lesson 5: Lists, Tuples, and Dictionaries

What is a List?

A list is a collection of ordered items. Lists are mutable, meaning you can change the items after the list has been created.

Creating a List

Lists are created by placing items inside square brackets, separated by commas:

fruits = ["apple", "banana", "cherry"]

Accessing List Items

You can access items in a list by referring to their index:

print(fruits[1])  # Output: banana

Modifying List Items

Since lists are mutable, you can change their content:

fruits[0] = "orange"  # Changes "apple" to "orange"

What is a Tuple?

A tuple is similar to a list but is immutable, meaning its values cannot be changed once assigned.

Creating a Tuple

Tuples are created by placing items inside parentheses:

coordinates = (4, 5)

Accessing Tuple Items

You can access items in a tuple by referring to their index, just like lists:

print(coordinates[0])  # Output: 4

What is a Dictionary?

A dictionary is a collection of key-value pairs. Dictionaries are unordered, meaning the items do not have a fixed order.

Creating a Dictionary

Dictionaries are created by placing key-value pairs inside curly braces:

person = {"name": "John", "age": 25}

Accessing Dictionary Values

You can access a value in a dictionary by referring to its key:

print(person["name"])  # Output: John

Modifying Dictionary Values

Since dictionaries are mutable, you can modify their values:

person["age"] = 26  # Changes the age to 26

  • Create a tuple of two numbers representing the coordinates of a location (x, y) and print the x coordinate.
