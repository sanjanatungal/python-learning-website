Lesson 3: Control Flow Statements in Python

In this lesson, we will learn about Python's control flow statements, which allow us to make decisions in our code. These are the key elements we will focus on:

  • If statements: Used for conditional execution.
  • Elif statements: Used for multiple conditions.
  • Else statements: Used to execute code when no conditions are met.
  • Loops: For repeating actions with for and while.

If statement:

if condition:
    # Execute this block if condition is True

Elif and Else statements:

if condition1:
    # Execute if condition1 is True
elif condition2:
    # Execute if condition2 is True
else:
    # Execute if none of the conditions are True

For Loops: Used for iterating over a sequence (like a list or range).

for item in sequence:
    # Execute code for each item in sequence

While Loops: Used to repeat code as long as a condition is True.

while condition:
    # Execute code as long as condition is True
